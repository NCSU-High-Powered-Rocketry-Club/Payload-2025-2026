2025-2026 Payload Challenge


## To run the camera:

1. Make sure the camera was plugged in before you turned on
the raspberry pi. If not, reboot the pi with the camera plugged in: `sudo reboot`.

2. Run the following command from the root level of this project: `./scripts/run_cam.sh`

3. You should see frames being printed out in the terminal.
